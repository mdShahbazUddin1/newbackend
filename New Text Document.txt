 index.js file, we can access the arguments passed in the terminal using the process.argv array. 
The first two elements of this array are always the path to the Node.js executable and the path to the current script file.

javascript code

const arg1 = parseInt(process.argv[2]);
const arg2 = parseInt(process.argv[3]);

const sum = arg1 + arg2;
console.log(sum);
In this code, parseInt() is used to convert the string arguments to numbers. 
JavaScript will concatenate the strings instead of adding the numbers.

Assuming you saved this code in index.js, running the command node index.js 1 2 will output 3 in the terminal.
